Projects

Butterfly Waveform Visualizer

The Butterfly Waveform Visualizer is a Python program that generates a synchronized visual representation of audio input, forming a beautiful butterfly-shaped animation. The visualizer maps audio waveform data to a dynamic butterfly curve while applying vibrant, shifting colors to circles drawn in real time.

The visualization runs as the audio plays, with features such as adjustable circle counts and smooth color gradients that rotate across the spectrum over time.

Field Trip

Field Trip is a React-based application for building and exploring immersive Virtual Field Environments (VFEs). This was a capstone project that I was a contributor on with a team of 5 others. My feature addition was the hotspot sidebar.

This feature adds a sidebar to the photosphere viewer that displays a hierarchical, collapsible list of all hotspots within each photosphere, allowing users to easily browse, expand, and select hotspots—including nested ones—across the virtual field trip. The sidebar enhances navigation by letting users quickly center the viewer on any hotspot, and provides a clear overview of the spatial structure and relationships between hotspots within and across photospheres. This improves usability for both exploration and editing workflows.
